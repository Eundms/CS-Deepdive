## 팩토리 메서드 패턴
객체를 만들기 위한 **인터페이스는 정의하고**, 실제 생성은 **하위 클래스**가 담당

```java
abstract class Product {}

class ConcreteProductA extends Product {}
class ConcreteProductB extends Product {}

abstract class Creator {
    public abstract Product createProduct();
}

class ConcreteCreatorA extends Creator {
    public Product createProduct() {
        return new ConcreteProductA();
    }
}

```


## 추상 팩토리 패턴 
서로 관련 있는 객체들을 **일관되게 생성**하기 위한 인터페이스 제공 

```java
interface Button {
    void click();
}

interface Checkbox {
    void check();
}

// Concrete 제품들
class MacButton implements Button {
    public void click() { System.out.println("Mac Button"); }
}

class MacCheckbox implements Checkbox {
    public void check() { System.out.println("Mac Checkbox"); }
}

class WindowsButton implements Button {
    public void click() { System.out.println("Windows Button"); }
}

class WindowsCheckbox implements Checkbox {
    public void check() { System.out.println("Windows Checkbox"); }
}

// 추상 팩토리
interface GUIFactory {
    Button createButton();
    Checkbox createCheckbox();
}

// 구체 팩토리
class MacFactory implements GUIFactory {
    public Button createButton() { return new MacButton(); }
    public Checkbox createCheckbox() { return new MacCheckbox(); }
}

class WindowsFactory implements GUIFactory {
    public Button createButton() { return new WindowsButton(); }
    public Checkbox createCheckbox() { return new WindowsCheckbox(); }
}

```
