# 편집기
- 행 단위 편집기 (한 줄 단위로 입력) : ed, ex, sed
- 화면 단위 편집기 (화면 단위로 입출력) : vi(vim), emacs, nano
- GUI 편집기 : gedit

## 모드형 편집기 vs 비모드형 편집기

### 모드형 편집기
- 입력 모드와 명령 모드가 구분
- 동일한 글자라도 입력 모드에서는 글자, 명령 모드에서는 명령어
- vi가 모드형 편집기

### 비모드형 편집기
- 입력 모드와 명령 모드가 구분되어 있지 않음
- 편집 기능을 CTRL이나 ALT 같은 특수 키 또는 메뉴로 제공
- nano가 비모드형 편집기

# vim

설치: `sudo apt install vim`

## 동작 모드
- 명령모드 → 입력모드 : `i, I, a, A, o, O`
  - `i` : 현재 커서 위치 앞에 입력 시작
  - `I` : 현재 줄의 맨 앞에서 입력 시작
  - `a` : 현재 커서 다음 위치에서 입력 시작
  - `A` : 현재 줄의 맨 끝에서 입력 시작
  - `o` : 현재 줄 다음 줄을 새로 만들고 입력 시작
  - `O` : 현재 줄 위에 새 줄 추가 후 입력 시작
- 입력모드 → 명령모드 : `ESC`
- 명령모드에서 저장 및 종료 : `ZZ`
- 마지막 행 모드 전환 : `: / ?`
- 마지막 행 모드에서 명령 모드 : `Enter` 또는 `ESC`
- 저장하지 않고 종료 : `:q` 또는 `:q!`
- 저장하고 종료 : `:wq` 또는 `:wq!`

## 커서 이동
- `k` / `j` : 한 행 위/아래 이동
- `l` / `h` : 한 글자 앞/뒤
- `^` 또는 `0` / `$` : 현재 행의 맨 앞/맨 뒤로 이동
- `-` / `+` 또는 `ENTER` : 다음 행의 처음으로 이동
- `H` / `M` / `L` : 화면의 맨 위/중간/아래로 이동
- `w` / `e` : 다음 단어의 처음/마지막 글자로 이동
- `b` : 앞 단어의 첫 글자로 이동

## 화면 이동
- `CTRL + u` : 반 화면 위로
- `CTRL + d` : 반 화면 아래로
- `CTRL + b` : 한 화면 위로
- `CTRL + f` : 한 화면 아래로
- `CTRL + y` : 한 행 위로
- `CTRL + e` : 한 행 아래로

## 특정 행 이동
- `Shift + G` : 파일의 마지막 행으로 이동
- `행번호 Shift + G` : 지정한 행 번호로 이동
- `:행번호` : 지정한 행 번호로 이동
- `:$` : 마지막 행으로 이동

## 수정
- `r` : 커서 위치 글자를 다른 글자로 수정
- `cw`, `#cw` : 커서부터 현재 단어 끝까지 수정
- `s`, `#s` : 커서부터 ESC를 누를 때까지 수정
- `cc` : 현재 행 전부 수정
- `C` : 커서부터 행의 끝까지 수정

## 삭제
- `x`, `#x` : 글자 삭제
- `dw`, `#dw` : 단어 삭제
- `dd`, `#dd` : 행 삭제
- `Shift + D` : 커서부터 행의 끝까지 삭제

## 명령 취소
- `u` : 명령을 취소
- `U` : 해당 행에서 수행한 모든 명령 취소
- `:e!` : 마지막 저장한 내용 이후 버리고 새로 작업

## 복사/잘라내기/붙여넣기
- `yy`, `#yy` : 행 복사
- `P` : 커서 위치 아래에 붙이기
- `p` : 커서 위치 위에 붙이기
- `dd`, `#dd` : 행 잘라내기

## 버퍼
- 복사나 잘라내기 시 내용을 저장하는 공간
- 언네임드 버퍼 : 하나의 내용만 저장
- 네임드 버퍼 : 이름을 붙여 여러 개 생성 가능
  - 만들기: `"문자나숫자 + 복사나잘라내기`
  - 붙여넣기: `"버퍼이름 + p`
