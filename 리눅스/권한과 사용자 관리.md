# sudo와 su
- `sudo` : 관리자의 권한을 빌려서 명령을 실행 (관리자 비밀번호 입력 필요)
- `su 계정` : 현재 계정의 환경 변수를 유지하면서 다른 계정으로 전환
- `su -계정` : 새로운 계정의 환경 변수를 유지하면서 다른 계정으로 전환
- `su` : root로 전환
