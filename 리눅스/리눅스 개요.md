# 리눅스 개요

## 리눅스 종류
- Debian 계열 : Ubuntu
- Slackware 계열 : Suse
- Redhat 계열 : Fedora, CentOS(Rocky), Redhat

## 리눅스 구조
- Kernel : Linux의 핵심, 컴퓨터의 모든 자원 초기화 및 제어, 장치 관리
- Shell : 명령어 해석기
- Application

## 가상화
- 하나의 물리적 컴퓨터에서 여러 운영체제를 동시에 실행할 수 있게 하는 기술
- 하나의 컴퓨터를 여러 개처럼 가상으로 쪼개서 사용하거나 여러 대를 묶어서 하나의 자원처럼 사용
- 가상화 툴 : VMWare, VirtualBox, Mac에서는 UTM
